## Про нас

- **Static Generation** - це метод попереднього рендерингу, який генерує HTML на **час збірки**. Попередньо рендерований HTML потім _перевикористовується_ при кожному запиті.
- **Server-side Rendering** - це метод попереднього рендерингу, який генерує HTML на **кожен запит**.

Важливо, що Next.js дозволяє **вибрати**, яку форму попереднього рендерингу використовувати для кожної сторінки. Ви можете створити "гібридний" додаток Next.js, використовуючи Static Generation для більшості сторінок і використовуючи Server-side Rendering для інших.
